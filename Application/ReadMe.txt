Assignment Description:
This project implements a FlowPanel system using X11 graphics. The FlowPanel manages TextAreas using a dynamic collection class TAArray. The project follows Object-Oriented Programming principles with proper encapsulation and memomry management


Header Files:
- TextArea.h – Defines the TextArea class for displaying text in a rectangle.
- FlowPanel.h – Defines the FlowPanel class for organizing TextAreas dynamically.
- TAArray.h – Implements a dynamic array of TextAreas.
- PanelArray.h – Implements a dynamic array of FlowPanels.

Source Files:
- TextArea.cc – This Implements TextArea methods.
- FlowPanel.cc – this Implements FlowPanel methods.
- TAArray.cc – This Implements the collection for managing TextAreas.
- PanelArray.cc – This Implements the collection for managing FlowPanels.

COmpilation Instructions:
- type "make" in the command line.
- run the program using "./a2"
